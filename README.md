# Taiwan Air Quality Dashboard
> Dashboard ini memvisualisasikan kualitas udara di Taiwan berdasarkan data Air Quality Index (AQI), persebaran polutan, dan distribusi status kualitas udara dari dataset Taiwan Air Quality Data.

🌐 Demo: https://visdat-kelompok7.vercel.app

## Isi Dashboard
- Chart 1: AQI Daily Trend — menampilkan perubahan nilai AQI harian untuk melihat tren kualitas udara dari waktu ke waktu.
- Chart 2: AQI by County — menampilkan perbandingan nilai AQI berdasarkan wilayah di Taiwan.
- Chart 3: Pollutant Distribution — menampilkan distribusi jenis polutan udara seperti PM2.5, PM10, CO, NO2, SO2, dan O3.
- Fitur interaktif: tooltip pada chart untuk melihat detail nilai data.
- Animasi: animasi entrance pada chart saat dashboard dimuat.

## Sumber Data
- Nama dataset: Taiwan Air Quality Data 2016-2024
- URL sumber: https://www.kaggle.com/datasets/taweilo/taiwan-air-quality-data-20162024

## Cara Jalankan di Lokal
# Jalur A (static):
Buka index.html menggunakan Live Server di VS Code.
